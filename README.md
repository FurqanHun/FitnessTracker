## Get started

1.  Clone the repo:
    ```bash
    git clone [https://github.com/FurqanHun/FitnessTracker.git](https://github.com/FurqanHun/FitnessTracker.git)
    cd FitnessTracker
    ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
   npx expo start
   ```

4.  Run:
    * Scan QR code with **Expo Go** (Android/iOS).
    * Or press `a` for Android Emulator.
